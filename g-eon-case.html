The Challenge

Design a mobile streaming ecosystem from scratch — covering live stream creation, viewer interactions, monetization, and creator tools — all in a single coherent mobile experience.

Three distinct user roles (viewer, agent, advertiser) needed different workflows while sharing the same app. Complex features like AR stream overlays, wallet transactions, tournament brackets, and streamer ranking systems had to feel intuitive on mobile.

The Solution

Designed the full mobile app experience across all user roles — with clear navigation patterns that adapted to each role's context. Viewers watch, donate, and join tournaments; streamers go live, manage donations, track rankings; advertisers set up and manage campaigns.

Designed the AR integration layer for live streams — interactive overlays that kept the viewing experience immersive without cluttering the stream. Also designed the wallet and donation flows for fast, low-friction transactions.

Designed the brand identity and logo for G-Eon alongside the product.

The Process

Worked with a team of 5 developers and 1 project manager. Covered the full design lifecycle: user research, flow mapping, wireframes, UI design, and prototyping. Regular syncs with developers to validate technical feasibility. Iterated on complex features like the AR overlay system and tournament UI based on team feedback.
